core package summary

core

Package includes modules that described core methods and options for doing actions related to

core.api

Package provides the core ApiClient class for interacting with the web API.

core.api.api_client

APIClient module as class provides a central interface for interacting with web APIs.

core.api.api_client_fixtures

Provides a fixtures for a pre-configured APIClient instance to interact with back-end and front-end API.

core.api.backend_api_points

Module as class to define the backend API endpoints as constants.

core.api.frontend_api_points

Module as class to define the frontend API endpoints as constants.

core.data.data_models

Package contains data models used API related to tested resources.

core.data.data_models.api_header_data_models

Represents HTTP headers for API requests.

core.data.data_models.api_login_data_models

This module defines the LoginCredentials dataclass, which is used to represent and handle user login credentials.

core.data.data_models.back_api_auth_data_models

A class representing the authentication payload for an API.

core.data.data_models.front_api_booking_object_data_model

A class representing the check-in and check-out dates for a booking.

core.data.data_models.front_api_bookingID_list_data_model

Class represents a payload containing a booking ID.

core.data.data_models.front_api_room_details_data_models

Class represents the details of a front room.
