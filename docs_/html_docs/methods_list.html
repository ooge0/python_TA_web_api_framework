core.api

Package provides the core ApiClient class for interacting with the web API.

The ApiClient class handles tasks like:

  • Sending API requests and receiving responses.

  • Managing authentication tokens.

  • Handling errors and exceptions.

Use this class for all interactions with the backend API of your application.

See Modules summary list

core.data

Package is responsible for managing and generating data models and factories used throughout the application, including test data for APIs, user credentials, and booking systems. This package is divided into multiple submodules that organize different types of data handling.
